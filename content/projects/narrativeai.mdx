# NarrativeAi

NarrativeAi is an interactive, text-based adventure game built with Next.js and OpenAI's GPT language model. The game allows users to create their own dynamically generated storylines based on popular TV shows, books and movies.

## Features

- Choose from any popular TV show, book or movie and select a character to play as.
- Begin an immersive, interactive story based on your selections.
- Continue the story by making decisions or performing actions as the chosen character.
- Powered by OpenAI's GPT language model, the game offers an engaging and dynamic text-based adventure.
