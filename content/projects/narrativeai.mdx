# NarrativeAi

NarrativeAi is an interactive, text-based adventure game built with Next.js and OpenAI's GPT language model. The game allows users to create their own adventures based on popular TV shows and movies.

## Features

- Choose from any popular TV show or movie and select a main character to play as.
- Begin an immersive, interactive story based on the chosen show or movie.
- Continue the story by making decisions or performing actions as the chosen character.
- Powered by OpenAI's GPT-3, the game offers an engaging and dynamic text-based adventure.

## Installation

To get started with NarrativeAi, clone the repository and install the dependencies.

`git clone https://github.com/arinewhouse/NarrativeAi.git`
`cd NarrativeAi`
`npm install`

## Setup

You will need to create a .env file in the root directory and add your OpenAI API key:

`OPENAI_API_KEY = "your-openai-api-key"`

## Running the Application

To start the application, run the following command in your terminal:

`npm run dev`

Then, open your browser and navigate to http://localhost:3000 to start playing the game.
